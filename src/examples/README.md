# Example Python Files

This folder contains example Python code files that can be loaded into the editor.

## Usage

These examples are loaded automatically when you click the "Load Sample" button in the editor. By default, `blink_led.py` is loaded when the page first opens.

## Adding New Examples

To add a new example:

1. Create a `.py` file in this directory
2. Write your Python/MicroPython code
3. Add a new entry for your file in `examples.json`
